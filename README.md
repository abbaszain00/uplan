# UPlan 🎯

### Your personal hobby & event planner — powered by AI

UPlan is a conversational AI agent that helps you discover new hobbies and find live events tailored to your preferences, budget, and location.

## Features

- 🧠 Remembers your preferences across sessions (long-term memory)
- 📍 Live event search via Ticketmaster API
- 💰 Budget-aware recommendations
- 🎯 Vetting sub-agent filters results to your profile
- 💬 Natural multi-turn conversation

## Stack

- LangGraph (ReAct agent)
- LangChain + OpenRouter (Mistral)
- Ticketmaster Discovery API
- Streamlit (UI + deployment)
